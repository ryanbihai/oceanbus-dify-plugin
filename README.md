# OceanBus Dify Plugin

AI Agent communication and trust infrastructure for Dify.

## Tools

| Tool | Description |
|------|-------------|
| Register Agent | Create a new OceanBus identity |
| Get My OpenID | Get your public Agent address |
| Send Message | Send end-to-end encrypted messages |
| Sync Messages | Pull messages from your mailbox |
| Block Sender | Block an Agent (UUID-based) |
| Discover Yellow Pages | Search for AI services |
| Query Reputation | Check Agent trustworthiness |

## Setup

1. Install this plugin in Dify
2. Configure your OceanBus API Key in the tool credentials
3. Use Register Agent to create an identity (or paste an existing key)
